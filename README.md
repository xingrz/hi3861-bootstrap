hi3861-bootstrap
==========

Minimal code that can boot on Hi3861.

The prebuilt objects and header files are taken from HiHope_WiFi-IoT_Hi3861SPC025_LiteOS_SDK.zip.

## Requirements

* riscv32-unknown-elf-gcc
* HiBurn

## Build

```sh
./build.sh
```
